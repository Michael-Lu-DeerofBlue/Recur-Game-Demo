
This asset contains 2 versions of Volumetric Fog & Mist.
Please use the appropriate package according to the rendering pipeline of your project:

- Volumetric Fog & Mist for the Universal Rendering Pipeline (URP) -> folder /URP.
- Volumetric Fog & Mist for the Builtin Pipeline -> folder /Builtin.

Once imported, please refer to the documentation folder included for additional setup instructions.

- Dynamic Fog & Mist can be downloaded from the Asset Store:
https://assetstore.unity.com/packages/vfx/shaders/fullscreen-camera-effects/dynamic-fog-mist-2-48200
